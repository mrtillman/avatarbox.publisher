# aws-update-gravatar

An AWS Lambda function to update your Gravatar icon

---

## Installation

```sh
$ git clone https://github.com/mrtillman/aws-update-gravatar.git
$ cd aws-update-gravatar
$ npm install
```

## Scripts

```sh
# zip code + dependencies
$ npm run zip

# deploy to AWS Lambda
$ npm run deploy
```
